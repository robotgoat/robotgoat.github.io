# My Blog

This is my blog that is built using AstroJS.
It is inspired by the Astrofy blog found here: https://github.com/manuelernestog/astrofy
Astrofy is outdated however, so I had to do a lot of modifications to get it to work with the latest Astro versions

The site can be used as a template for any of your needs.
Just delete everything in the `/src/content/blog/` folder.

## Features

### Autmatic Blog Page Generation

To create a new blog post:

`npm run createblog $NAME $DESCRIPTION`

Options: `d` - creates the markdown in its own directory.
